🔢 Giải Phương Trình Bậc Hai

📝 Đề bài
Giải phương trình: 2x² - 5x + 2 = 0
1
Xác định các hệ số
So sánh phương trình với dạng tổng quát ax² + bx + c = 0
a = 2, b = -5, c = 2
2
Tính Delta (Δ)
Sử dụng công thức: Δ = b² - 4ac
Δ = (-5)² - 4(2)(2) = 25 - 16 = 9
3
Kiểm tra dấu của Delta
Vì Δ = 9 > 0, phương trình có hai nghiệm phân biệt
√Δ = √9 = 3
4
Tính các nghiệm
Áp dụng công thức nghiệm: x = (-b ± √Δ) / 2a
x₁ = (-(-5) + 3) / (2×2) = 8/4 = 2
x₂ = (-(-5) - 3) / (2×2) = 2/4 = 0.5
✓ KẾT QUẢ
S = {0.5 ; 2}
