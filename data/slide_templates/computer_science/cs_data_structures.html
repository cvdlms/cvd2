Data Structures - Nền tảng của lập trình
Array (Mảng)
Tập hợp các phần tử cùng kiểu, lưu trữ liên tiếp trong bộ nhớ
Truy cập: O(1)
Tìm kiếm: O(n)
Stack (Ngăn xếp)
Cấu trúc LIFO (Last In First Out) - vào sau ra trước
Push/Pop: O(1)
Peek: O(1)
Queue (Hàng đợi)
Cấu trúc FIFO (First In First Out) - vào trước ra trước
Enqueue: O(1)
Dequeue: O(1)
Linked List
Chuỗi các node liên kết, mỗi node chứa data và pointer
Chèn/Xóa: O(1)
Truy cập: O(n)
Tree (Cây)
Cấu trúc phân cấp với root, parent và child nodes
Tìm kiếm: O(log n)
Chèn: O(log n)
Graph (Đồ thị)
Tập hợp các đỉnh (vertex) và cạnh (edge) kết nối chúng
DFS/BFS: O(V+E)
Dijkstra: O(V²)
