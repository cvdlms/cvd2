🌐 Kiến Trúc Web Development

Full Stack - Từ giao diện đến cơ sở dữ liệu
Frontend (Client-side)
Giao diện người dùng - Những gì người dùng nhìn thấy và tương tác. Chạy trên trình duyệt web.
Công nghệ:
HTML5 CSS3 JavaScript React Vue.js Angular Tailwind CSS
Backend (Server-side)
Xử lý logic nghiệp vụ, API, xác thực, và giao tiếp với database. Chạy trên server.
Công nghệ:
Node.js PHP Python Java Express Laravel Django
Database (Cơ sở dữ liệu)
Lưu trữ và quản lý dữ liệu ứng dụng. Có thể là SQL (quan hệ) hoặc NoSQL (phi quan hệ).
Công nghệ:
MySQL PostgreSQL MongoDB Redis SQLite Firebase
