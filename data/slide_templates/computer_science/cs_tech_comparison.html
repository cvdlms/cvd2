⚔️ So Sánh: SQL vs NoSQL

VS
SQL
Relational Database
  • Cấu trúc bảng cố định với schema
  • Dữ liệu có quan hệ rõ ràng
  • Ngôn ngữ truy vấn SQL chuẩn
  • ACID compliance (tính toàn vẹn)
  • Phù hợp: Banking, CRM, ERP
  • VD: MySQL, PostgreSQL, SQL Server
NoSQL
Non-Relational Database
  • Cấu trúc linh hoạt, schema-less
  • Dữ liệu dạng document, key-value
  • Truy vấn theo từng DB riêng
  • BASE (eventual consistency)
  • Phù hợp: Social media, IoT, Big data
  • VD: MongoDB, Redis, Cassandra
