🔄 Vòng Đời Phát Triển Phần Mềm

SDLC - Software Development Lifecycle
1
Planning
(Lập Kế Hoạch)
Xác định mục tiêu, phạm vi, nguồn lực và lịch trình dự án.
• Phân tích yêu cầu
• Đánh giá khả thi
• Lập ngân sách
2
Design
(Thiết Kế)
Thiết kế kiến trúc hệ thống, cơ sở dữ liệu và giao diện.
• System architecture
• Database schema
• UI/UX mockups
3
Development
(Phát Triển)
Lập trình viên viết code để xây dựng các tính năng.
• Coding
• Version control
• Code review
4
Testing
(Kiểm Thử)
Kiểm tra lỗi, đảm bảo chất lượng và hiệu năng.
• Unit testing
• Integration testing
• User acceptance
5
Deployment
(Triển Khai)
Đưa sản phẩm lên môi trường production, người dùng sử dụng.
• Release planning
• Server setup
• User training
6
Maintenance
(Bảo Trì)
Sửa lỗi, cập nhật tính năng và hỗ trợ người dùng.
• Bug fixes
• Updates
• Performance tuning
📋 Phương Pháp Phát Triển Phổ Biến
Waterfall
Tuần tự, từng bước một, phù hợp dự án rõ ràng
Agile
Linh hoạt, lặp đi lặp lại, thích nghi thay đổi
Scrum
Sprint 2-4 tuần, daily standup, review meeting
